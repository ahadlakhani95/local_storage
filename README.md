Flutter Local Storage App
This Flutter application demonstrates local data storage using SharedPreferences.
The app allows users to store, retrieve, update, and delete data locally on the device.
📱 Features
•	Store data locally using SharedPreferences 
•	Retrieve saved data 
•	Update existing data 
•	Delete stored data 
•	Simple and user-friendly UI 
🛠️ Technologies Used
•	Flutter 
•	Dart 
•	SharedPreferences (Local Storage) 
•	Android Studio / VS Code

🚀 How the App Works
1.	User enters text in the TextField 
2.	Click Save Data → Data stored locally 
3.	Click Read Data → Data displayed on screen 
4.	Click Update Data → Existing data updated 
5.	Click Delete Data → Data removed from local storage
